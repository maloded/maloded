# Frontend App 🚀

A production-grade React application built with Feature-Sliced Design (FSD) architecture. Includes full testing setup, Storybook, i18n, and custom Webpack/Vite configuration.

## Features

- 🏗️ **Architecture** — Feature-Sliced Design (FSD)
- 🌐 **i18n** — Multi-language support via i18next
- 🎨 **UI** — Custom component library with Storybook
- 🧪 **Testing** — Unit (Jest), E2E (Cypress), Visual regression (Loki)
- ⚡ **Build** — Webpack 5 + Vite support

## Tech Stack

- **Framework:** React 18 + TypeScript
- **State Management:** Redux Toolkit
- **Routing:** React Router v6
- **HTTP Client:** Axios
- **Styling:** SCSS + CSS Modules
- **i18n:** i18next + react-i18next
- **Animation:** @react-spring/web + @use-gesture/react
- **Build:** Webpack 5 / Vite 4
- **Testing:** Jest + Testing Library + Cypress + Loki
- **Documentation:** Storybook 7
- **Linting:** ESLint (Airbnb) + Stylelint + Prettier
